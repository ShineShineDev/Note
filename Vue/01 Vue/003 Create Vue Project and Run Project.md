#### Create Vue Project With NPM (recommended way)

```js
// install latest version
> npm init vue@latest

// install version 3
> npm init vue@3 

// install version 2
> npm init vue@2

// go to project folder
> cd your_project_folder

// install packages
> npm install

// run 
> npm run dev

// build
> npm run build
```





#### Create Vue Project With Vue CLI UI

```js
> vue ui
```
